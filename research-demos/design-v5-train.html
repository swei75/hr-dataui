设计说明

  • 覆盖维度：6/6 完整（5 项目气泡 / 100% 部门 8 / 末位 3 / 资格考 3 数字）
  • 与 M-1~M-5 差异：M-3 用 4 节点 linear flow 串接；本版改用 X-Y 两轴 quadrant，可一眼看出"哪些项目要立刻抓、哪些可以排期"；M-4 是 13 节点分类树，本版是 2D 平面定位，无层级。
  • 新增 viz 类型：① 2D 四象限矩阵（quadrant matrix · X=影响力 Y=紧急度）② 气泡图（bubble plot · 面积=期数）③ 责任主体 chip 矩阵（chip-matrix × 3，三栏分组）
  • 信息密度：高 — 5 项目分层 + 8+3 责任主体 + 3 个资格考数字
  • 读图速度：3 秒 — 红色气泡立即吸引至"紧急核心"，chip 区一句话锁定责任主体
