设计说明

  • 覆盖维度：12/12 完整（中层总 / 总行 / 经营 / 正 / 副 / 副总级一把手 / 副总 / 总助 / 经理 / 兼任 / 已配 / 空缺）
  • 与 M-1 差异：M-1 是 6 个独立 KPI；v5 是 4 列层级 + 12 条流向带，强调路径
  • 与 M-2 差异：M-2 是 stacked-bar 占比卡；v5 是流图，关注起点→终点
  • 与 M-3 差异：M-3 v2 是流程图（4 节点 + 箭头，强调步骤）；v5 是桑基（曲带宽度映射流量）
  • 新增 viz 类型：① 桑基曲线（cubic bezier ribbon） ② 4 阶段流向（总→机构→职级→状态） ③ 流量宽度编码（带粗 = 流量大） ④ 多列节点（11 节点跨 4 列）
  • 信息密度：高 — 11 节点 + 12 条流 + 节点内 2-3 字段 = 50+ 数字
  • 读图速度：5-7 秒 — 需要追踪曲带，理解逐级拆分
  • 取舍：纯 CSS 无法画曲线，用 SVG path + cubic bezier 实现；窄屏（<980px）切换为纵向排列并隐藏流层
