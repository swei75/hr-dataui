设计说明

  • 覆盖维度：12/12 完整（中层总 / 正 / 副 / 副总级一把手 / 副总 / 总助 / 经理 / 兼任 / 总行 / 经营 / 已配 / 空缺）
  • 与 M-1 差异：M-1 是 6 个等权数字；v6 用金字塔表示"顶层窄 / 底层宽"的等级结构
  • 与 M-2 差异：M-2 是 5 段 stacked-bar 占比卡（横向延展）；v6 是垂直 4 层金字塔（顶端小 / 底端大）
  • 与 M-3 差异：M-3 v1 是时间轴（横向扫月）；v6 是层级（垂直下钻）
  • 新增 viz 类型：① 4 层金字塔（pyramid by width） ② 配对层（L2 正副职并列成对） ③ 子类 5 元拆分（L3 副总级/副总/总助/经理/兼任） ④ 颜色编码层级
  • 信息密度：高 — 4 层 × 多节点 + 3 补充卡 = 30+ 数字
  • 读图速度：4 秒 — 自顶向下扫读，金字塔视觉即"等级感"
