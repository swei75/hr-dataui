设计说明

  • 覆盖维度：6/6 完整（培训总量 / 重点项目 / 执行率 / 100% 部门 / 末位部门 / 资格考）
  • 与 M-1~M-5 差异：M-1 是 6 静态 KPI 卡（散点数字）/ M-2 是 stacked-bar 客户网格（占比堆积）/ M-3 是 gauge ring + 颁奖台（环形与阶梯）/ M-4 是 13 节点分类树（层级结构）/ M-5 是大数字 + sparkline 财务卡（趋势）。本版把 6 维压成单张 dense table，行内 mini bar 做完成率，行末 rank-bar 做排名，紧凑呈现"对比 + 排序"信息密度。
  • 新增 viz 类型：① 多列对比矩阵（5 列 dense table：子方向 / 当前值 / 目标值 / 完成率 / 排名）；② 行内 mini horizontal bar（颜色按 ≥80%/50-80%/<50% 分档）；③ 反向排名条（行末 rank-bar，排名越靠前条越长）；④ mini-tag chip list（100% 部门绿底，末位部门红底 ⚠）
  • 读图速度：4 秒 — 表头五行将"现状 / 目标 / 完成 / 排名"一次性扫完，末位 3 部门和 100% 部门再用 chip 集中列出，便于快速锁定优劣两端。
