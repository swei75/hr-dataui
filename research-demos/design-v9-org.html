设计说明

  • 覆盖维度：12/12 完整（中层总 / 总行 / 经营 / 正 / 副 / 副总级一把手 / 副总 / 总助 / 经理 / 兼任 / 已配 / 空缺 / 市管 / 高管 / 纪检 / 调整情况）
  • 与 M-1 差异：M-1 是 6 个等权 KPI；v9 是 7 行 3 重柱状图（21 根柱），直方对比
  • 与 M-2 差异：M-2 是 stacked-bar 占满 100% 宽度；v9 是 3 根独立柱并排（绝对值并列对比，非堆叠）
  • 与 M-3 差异：M-3 v1 是 gauge 单色进度 + 5 月时间轴；v9 是每行 3 重柱，无时间维度
  • 新增 viz 类型：① 3 重柱（triple-bar grouped，绝对值并排） ② 三角对比（总数 vs 已配 vs 空缺一图对照） ③ 柱顶数字 + 渐变填充 ④ 警示色命名（纪检行"3/3"红色高亮）
  • 信息密度：高 — 7 行 × 3 柱 + 数字 = 28+ 字段
  • 读图速度：4 秒 — 7 行依次扫读，红柱高度即缺口比例
