## P0-B delta 自动计算（2 任务）

### T5：实现 `composer.calc_delta()` 函数

**文件**：`composer.py`（新增函数）

**改动**：
```python
def calc_delta(current: float, prev: float | None) -> dict | None:
    if prev is None or prev == 0:
        return None
    diff = current - prev
    pct = diff / abs(prev) if prev != 0 else 0
    direction = "up" if diff > 0 else "down" if diff < 0 else "flat"
    return {"abs": diff, "pct": pct, "direction": direction, "display": f"{'+' if diff > 0 else ''}{pct*100:.1f}%"}
```

**Success Criteria**：
- 输入 (100, 80) → {"abs": 20, "pct": 0.25, "direction": "up", "display": "+25.0%"}
- 输入 (50, None) → None
- 输入 (50, 0) → None

**验证**：
```bash
python -c "from composer import calc_delta; print(calc_delta(100, 80)); print(calc_delta(50, None))"
```
